    prompt = f"""魹ｽ欵ｫ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽﾑｯ魹ｽ魹ｽﾐｴﾎｪ 3 魹ｽ魹ｽ魹ｽ魹ｽﾍｬ魹ｽﾇｶﾈｵ魹ｽﾍｬ魹ｽ魹ｽ魹ｽﾑｯ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽﾙｻ魹ｽ魹ｽﾊ｡魹ｽ

## 魹ｽ魹ｽ魹ｽ魹ｽ魹ｽﾅｻ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽﾋｵ魹ｽ魹ｽﾋｴ魹ｽ魹ｽ魹ｽﾊｾ魹ｽ魹ｽ

魹ｽ魹ｽ魹ｽ魹ｽ魹ｽﾏｷ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽﾎｪﾒｻ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽﾅｻ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽﾂｷ魹ｽ魹ｽ

```python
"""
RAG 魹ｽﾙｻ魹ｽ魹ｽ魹ｽ魹ｽﾅｻ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽﾏｼ魹ｽ魹ｽ魹ｽ + 魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ徸ｽｰ魹ｽ
魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽpip install langchain langchain-openai faiss-cpu rank-bm25 FlagEmbedding
"""
from langchain_openai import OpenAIEmbeddings, ChatOpenAI
from langchain.vectorstores import FAISS
from langchain.retrievers import BM25Retriever, EnsembleRetriever
from langchain.text_splitter import RecursiveCharacterTextSplitter
from langchain.schema import Document
from FlagEmbedding import FlagReranker


class OptimizedRAGRetriever:
    """魹ｽﾅｻ魹ｽ魹ｽ魹ｽ RAG 魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽﾐｷ魹ｽ 魹ｽ魹ｽ 魹ｽ魹ｽﾏｼ魹ｽ魹ｽ魹ｽ 魹ｽ魹ｽ 魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ"""

    def __init__(self, documents: list[str]):
        self.llm = ChatOpenAI(model="gpt-4o", temperature=0)
        self.embeddings = OpenAIEmbeddings(model="text-embedding-3-large")
        self.reranker = FlagReranker('BAAI/bge-reranker-v2-m3', use_fp16=True)

        # Step 1: 魹ｽﾝｹ魹ｽ魹ｽﾐｷﾖ｣魹ｽ魹ｽ魹ｽ魹ｽﾘｵ魹ｽ魹ｽ魹ｽ
        splitter = RecursiveCharacterTextSplitter(
            chunk_size=600,
            chunk_overlap=120,
            separators=["¥n¥n", "¥n", "魹ｽ魹ｽ", "魹ｽ魹ｽ", "魹ｽ魹ｽ", "魹ｽ魹ｽ", " ", ""],
        )
        self.chunks = [
            Document(page_content=text, metadata={"source": f"doc_{i}"})
            for i, text in enumerate(splitter.split_text("¥n¥n".join(documents)))
        ]

        # Step 2: 魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ + BM25 魹ｽ魹ｽ魹ｽ魹ｽ
        self.vector_store = FAISS.from_documents(self.chunks, self.embeddings)
        self.bm25_retriever = BM25Retriever.from_documents(self.chunks)
        self.bm25_retriever.k = 30

    def query_rewrite(self, question: str) -> list[str]:
        """魹ｽ魹ｽﾑｯ魹ｽ魹ｽﾐｴ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽﾉｶ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽﾑｯ"""
        prompt = f"""魹ｽ魹ｽ魹ｽ魹ｽ魹ｽﾂｲ魹ｽﾑｯ魹ｽ魹ｽﾐｴﾎｪ3魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽﾍｬ魹ｽ魹ｽ魹ｽ魹ｽ驒ｻﾍｬ魹ｽﾄｱ魹ｽ魹ｽ螢ｬﾃｿ魹ｽ魹ｽﾒｻ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽﾒｪ魹ｽ魹ｽﾅ｡魹ｽ
魹ｽ魹ｽﾑｯ魹ｽ魹ｽ{question}"""
        response = self.llm.invoke(prompt).content
        queries = [question] + [q.strip() for q in response.strip().split("¥n") if q.strip()]
        return queries[:4]  # 魹ｽ魹ｽ魹ｽ 4 魹ｽ魹ｽ魹ｽ魹ｽﾑｯ

    def hybrid_retrieve(self, query: str, top_k: int = 30) -> list[Document]:
        """魹ｽ魹ｽﾏｼ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ + BM25"""
        vector_retriever = self.vector_store.as_retriever(
            search_kwargs={"k": top_k}
        )
        ensemble = EnsembleRetriever(
            retrievers=[vector_retriever, self.bm25_retriever],
            weights=[0.6, 0.4],
        )
        return ensemble.get_relevant_documents(query)

    def rerank(self, query: str, candidates: list[Document], top_n: int = 5) -> list[Document]:
        """Cross-Encoder 魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ"""
        pairs = [[query, doc.page_content] for doc in candidates]
        scores = self.reranker.compute_score(pairs)
        if isinstance(scores, float):
            scores = [scores]
        ranked = sorted(zip(candidates, scores), key=lambda x: x[1], reverse=True)
        return [doc for doc, _ in ranked[:top_n]]

    def retrieve(self, question: str, top_n: int = 5) -> list[Document]:
        """魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽﾂｷ魹ｽ魹ｽ魹ｽ魹ｽﾐｴ 魹ｽ魹ｽ 魹ｽ魹ｽﾏｼ魹ｽ魹ｽ魹ｽ 魹ｽ魹ｽ ﾈ･魹ｽ魹ｽ 魹ｽ魹ｽ 魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ"""
        # 1. 魹ｽ魹ｽﾑｯ魹ｽ魹ｽﾐｴ
        queries = self.query_rewrite(question)

        # 2. 魹ｽ魹ｽ魹ｽﾑｯ魹ｽ魹ｽﾏｼ魹ｽ魹ｽ魹ｽ
        all_candidates = []
        seen = set()
        for q in queries:
            for doc in self.hybrid_retrieve(q):
                content_hash = hash(doc.page_content[:100])
                if content_hash not in seen:
                    seen.add(content_hash)
                    all_candidates.append(doc)

        # 3. 魹ｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽﾈ｡ Top-N
        return self.rerank(question, all_candidates, top_n)


# ﾊｹ魹ｽ魹ｽﾊｾ魹ｽ魹ｽ
if __name__ == "__main__":
    docs = ["...魹ｽ魹ｽ魹ｽﾖｪﾊｶ魹ｽ魹ｽ魹ｽﾄｵ魹ｽ..."]
    retriever = OptimizedRAGRetriever(docs)
    results = retriever.retrieve("魹ｽ邀｣魹ｽﾏｽ魹ｽ魹ｽ魹ｽ魹ｽ魹ｽﾃｴ魹ｽ魹ｽ魹ｽﾉ｣魹ｽ")
    for r in results:
        print(r.page_content[:80], r.metadata)
```

# CmCtrlCmdProcPlug 模块功能说明

## 1. 模块概述

CmCtrlCmdProcPlug 是一个控制命令处理插件，负责处理各种控制命令的执行和管理，包括信号控制、列车控制、站台控制等功能。

## 2. 核心功能

- **命令处理**：处理各种控制命令的执行逻辑
- **信号控制**：信号机的开放、关闭等控制
- **列车控制**：列车的折返、跳停、扣车等控制
- **站台控制**：站台的停站时间调整、自动折返等控制
- **安全控制**：提供安全密码验证机制
- **命令管理**：命令的记录、查询和管理

## 3. 主要组件

| 组件名称 | 功能描述 |
|---------|---------|
| CommandBase | 命令基类，定义命令的基本接口 |
| CommandTSR | TSR命令处理，处理临时限速命令 |
| CommandZC | ZC命令处理，处理区域控制器命令 |
| DlgAddTsrDev | 添加TSR设备对话框 |
| DlgPassword | 密码验证对话框 |
| DlgSetReback | 设置折返对话框 |
| DlgZcCmdSet | ZC命令设置对话框 |
| LockBar | 锁定工具栏，提供锁定功能 |
| MyToolBarEx | 自定义工具栏，提供扩展功能 |
| QButtonST | 自定义按钮，提供特殊样式 |
| SafeCmdState | 安全命令状态管理 |
| TSRCmdMgr | TSR命令管理器 |
| TaskTSR | TSR任务处理 |
| TaskZcCmd | ZC命令任务处理 |
| VC2Qt_cccp | VC到Qt的转换工具 |
| ZcCmdMgr | ZC命令管理器 |

## 4. 技术实现

- 基于Qt框架开发
- 实现了命令模式设计
- 支持密码安全验证
- 提供了丰富的对话框和UI组件
- 支持多任务并发处理

## 5. 应用场景

- 信号机的远程控制
- 列车的远程指挥和控制
- 站台设备的控制和管理
- 临时限速命令的处理
- 区域控制器命令的处理

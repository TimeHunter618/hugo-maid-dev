# UserManagerPlug 模块功能说明

## 1. 模块概述
UserManagerPlug是用户管理插件，负责系统中的用户管理功能。该模块主要实现用户的添加、编辑、删除、查询等功能，以及用户权限的管理和角色分配，为系统提供全面的用户管理支持。

## 2. 主要功能

### 2.1 用户管理
- 实现用户的添加、编辑和删除
- 支持用户信息的查询和搜索
- 提供用户列表的显示和管理

### 2.2 角色管理
- 实现角色的添加、编辑和删除
- 支持角色权限的分配和管理
- 提供角色列表的显示和管理

### 2.3 权限管理
- 管理系统中的权限资源
- 支持权限的分配和回收
- 提供权限的查询和验证

### 2.4 用户组管理
- 实现用户组的添加、编辑和删除
- 支持用户组的成员管理
- 提供用户组的权限分配

### 2.5 登录记录管理
- 记录用户的登录日志
- 支持登录记录的查询和统计
- 提供登录记录的导出功能

### 2.6 密码管理
- 支持用户密码的重置
- 实现密码强度的检查
- 提供密码策略的配置

## 3. 核心组件

### 3.1 用户管理对话框
- **QDlgUserManage.h**：用户管理对话框，用于管理系统中的用户
- **QDlgAddUser.h**：添加用户对话框，用于添加新用户
- **QDlgEditUser.h**：编辑用户对话框，用于编辑现有用户

### 3.2 角色管理对话框
- **QDlgRoleManage.h**：角色管理对话框，用于管理系统中的角色
- **QDlgAddRole.h**：添加角色对话框，用于添加新角色

### 3.3 用户组管理对话框
- **QDlgGroupManage.h**：用户组管理对话框，用于管理系统中的用户组
- **QDlgAddUserGroup.h**：添加用户组对话框，用于添加新用户组

### 3.4 登录记录对话框
- **QDlgLogonRecord.h**：登录记录对话框，用于查看用户的登录记录

### 3.5 工具类
- **Item.h/cpp**：列表项类，用于在列表中显示用户、角色等信息
- **MyCombox.h/cpp**：自定义组合框类，用于特殊的组合框需求

## 4. 技术实现

### 4.1 插件架构
- 实现了标准的插件接口
- 支持插件的动态加载和卸载

### 4.2 图形界面
- 基于Qt框架实现图形界面
- 使用Qt Designer设计对话框

- 支持继承和多态

### 4.4 数据管理
- 实现了用户、角色、权限等数据的管理
- 支持数据的持久化存储
- 提供数据的查询和统计功能

## 7. 工作流程

### 7.1 用户管理流程
1. 打开用户管理界面
2. 选择要执行的操作（添加、编辑、删除、查询）
3. 执行相应的操作
4. 系统更新用户数据
5. 刷新用户列表

### 7.2 角色管理流程
1. 打开角色管理界面
2. 选择要执行的操作（添加、编辑、删除、分配权限）
3. 执行相应的操作
4. 系统更新角色数据
5. 刷新角色列表

### 7.3 权限分配流程
1. 打开角色或用户组的权限分配界面
2. 选择要分配的权限
3. 保存权限分配
4. 系统更新权限数据

## 9. 使用说明

### 9.1 插件加载
- 通过插件管理器自动加载
- 在系统菜单中选择用户管理功能

### 9.2 操作流程

#### 9.2.1 用户管理
1. 打开用户管理界面
2. 点击"添加"按钮添加新用户
3. 选择现有用户，点击"编辑"按钮修改用户信息
4. 选择现有用户，点击"删除"按钮删除用户
5. 使用搜索功能查找特定用户

#### 9.2.2 角色管理
1. 打开角色管理界面
2. 点击"添加"按钮添加新角色
3. 选择现有角色，点击"编辑"按钮修改角色信息
4. 选择现有角色，点击"权限分配"按钮分配权限

#### 9.2.3 用户组管理
1. 打开用户组管理界面
2. 点击"添加"按钮添加新用户组
3. 选择现有用户组，点击"编辑"按钮修改用户组成员
4. 选择现有用户组，点击"权限分配"按钮分配权限

### 9.3 配置文件
- 使用系统默认配置
- 可通过配置文件调整用户管理参数

### 9.4 日志输出
- 输出用户管理相关日志
- 记录用户管理操作和权限变更
